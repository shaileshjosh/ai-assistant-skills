# Flutter Project Bootstrap

## When to use

Use this skill when:

- Creating a new Flutter project
- Initializing project architecture
- Setting up project dependencies
- Establishing project standards

## Instructions

Read and apply:

- architecture
- api-standards
- auth-standards
- data-storage
- caching
- unit-testing

Create:

lib/
├── core/
├── features/
├── shared/
└── main.dart

Configure:

- Riverpod
- Dio
- GoRouter
- Hive
- FlutterSecureStorage
- local_auth

Generate:

- AppConstants
- ApiConstants
- CacheConstants
- ApiClient
- SecureStorageService
- HiveStorageService

Verify project builds successfully.
